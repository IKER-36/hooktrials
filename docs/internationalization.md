# English and Spanish interface

HookTrials ships with complete English (`en`) and Spanish (`es`) interfaces for authentication, the
dashboard, product tour, in-product documentation, forms, feedback, evidence and public status
pages.

Use the **Language / Idioma** selector in the sidebar, authentication screen or public status page.
The choice is stored in the browser. A direct link can override it with `?lang=en` or `?lang=es`;
this is useful when linking from a translated website or sending a public status page to a specific
audience.

User-created names, payload data, hostnames, provider values and stored incident evidence are never
machine-translated. Product labels, validation feedback, dates and relative times follow the active
locale. English remains the source language used by contributors.
