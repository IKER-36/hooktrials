# HookTrials v0.11.3 — dark operational surfaces

HookTrials `v0.11.3` fixes the remaining light-theme leakage visible when the dashboard runs in dark
mode.

## Included

- dark semantic markers for Production Readiness passed, missing and warning states;
- dark monitor nominal and incident notices;
- readable monitor protocol, environment and interval chips;
- dark action buttons for monitor controls and public status pages;
- dark check-history rows and status-link controls.

This is a presentation-only patch. It preserves the existing Product/Lab separation and all delivery,
monitoring, persistence and self-hosting behaviour.

## Validation

- Prettier check for the changed stylesheet;
- strict web TypeScript check;
- production Vite build;
- CubePath health and four-origin smoke after promotion.
